เกณฑ์ AI แนะนำ — ใช้ Cost per Message (฿)

คำนวณจาก: ราคาขาย × (1 - margin%) × close rate

≤ ฿35 → เพิ่มงบ ≤ ฿42 → ต่อ ≤ ฿55 → รอดูก่อน > ฿55 → หยุด

💡 เกณฑ์นี้คำนวณจากข้อมูลจริง: ขาย ฿2,100 / margin 80% / close rate 10%

ROAS Override — ถ้า ROAS ดีพอ ให้ผ่านแม้ CPMsg จะแพง

ROAS จาก Meta pixel อาจไม่ครบ (Instagram Message ไม่นับ) ใช้เป็น signal เสริมเท่านั้น

ขั้นตอน (ใช้เวลา ~5 นาที):

1. ไปที่ Graph API Explorer

2. เลือก App ของคุณ (หรือ Meta App ไหนก็ได้ที่เชื่อมกับ Ad Account)

3. คลิก "Generate Access Token"

4. เพิ่ม Permissions: ads_read

5. Copy token มาใส่ในช่องด้านบน

Token นี้หมดอายุใน ~1 ชม. สำหรับใช้งานจริงให้ขอ Long-lived Token (60 วัน)

6. Ad Account ID → เปิด Ads Manager → ดู URL จะเห็น act_XXXXXXXXXX
